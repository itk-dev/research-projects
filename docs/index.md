## Projects

### [Climate Awareness Nudging](projects/climate-nudging/index.md)

**Status:** Draft | **Started:** April 2026

How can we nudge AI users to be more aware of the climate impact of their usage — without guilt-tripping or blocking workflows?

This project explores nudging strategies, UI concepts, CO2 estimates, and integration approaches for platforms like Open WebUI and Claude Code.

**Key deliverables:**

- [Research on AI energy consumption and CO2 estimates](projects/climate-nudging/co2-research.md)
- [17 nudging ideas across 4 categories](projects/climate-nudging/ideas.md)
- [Interactive UI mocks](projects/climate-nudging/index.md#ui-concepts-and-mocks) (leaf indicator, session ticker, usage dashboard)
- [Technical integration analysis](projects/climate-nudging/integration.md) for Open WebUI, Claude Code, and API layers
- [Guidance to hosting providers](projects/climate-nudging/co2-research.md#5-guidance-to-hosting-providers) on data transparency
